Frogjump

Frogjump is a remote control program for Google Maps for Android.

This allows (mostly) hands-free remote control of Google Maps for Android navigation, so that a driver can use their mobile phone with Google Maps while the passengers set the destination.

There is also support for as the crow flies navigation with GPS Status and Toolbox.

To start navigation, you must click a Google Maps or geo: link. Only Google Maps links in Five Eyes countries are supported right now.

Get it on the Google Play Store (beta)!

Please note, it may take up to an hour between opting in to the beta version of Frogjump, and it being available to install in the Play Store.

Supports

  • geo: links
  • Google Maps links in Five Eyes countries
  • Google Maps navigation by car, bike, foot, and avoiding toll roads.
  • GPS Status and Toolbox "radar" navigation

Caveats

  • Does not hook the "share" function inside Google Maps itself, because this doesn't use geo: URLs.
  • It isn't possible to see recent messages sent to the group.
  • It isn't possible to see who is in the group.
  • Only supports Google Maps. Other apps can be implemented where they have similar hands-free intents.
  • If navigation is in progress, Google Maps needs confirmation to change destination.
  • GCM registration is fragile. Rotate your device to retry the connection.
  • The UI is ugly.
